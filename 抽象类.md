## 概念
一个没有方法体的方法定义为抽象方法，如果类中有抽象方法则定义为 抽象类 
需要把类和方法加入 abstract
eg

    public abstract class Animal{
      public abstract void eat();
    }
    
## 特点
抽象类不能实例化，但是可以用多态的方法通过子类对象实例化 = 抽象类多态
抽象类的子类：要么重写抽象类中的所有抽象方法，要么是抽象类

## 成员特点
可以是成员变量：变量或者常量
构造方法：可以有，用于子类访问的时候，对父类数据初始化
成员方法：可以有抽象方法，限定子类必须完成某些动作，也可以有非抽象提高代码复用性
